前言 ​

执行 git commit 时，使用 Emoji，可凸显提交信息，易于区分和查找提交历史

Commit 格式 ​

提交信息遵从以下格式

html
:emoji:<type>(<scope>): <subject> 不超过 100 个字的摘要，首字母小写，使用祈使语气，句末不加标点
1
