TIP

本文以 master -> main 举例

重命名本地分支 ​

shell
$ git branch -m master main
1

将新分支同步到远端并设置 upstream 到分支 ​

shell
$ git push -u origin main
1

切换默认分支 ​

到 GitHub 的仓库设置中选择 main 为默认分支

删除旧分支 ​

shell
git push origin --delete master
1
