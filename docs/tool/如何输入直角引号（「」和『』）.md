   ![file](/images/如何输入直角引号（「」和『』）/option_2.png)

3. 选择词库和自学习

   ![file](/images/如何输入直角引号（「」和『』）/ciku.png)

   ![file](/images/如何输入直角引号（「」和『』）/add_phrase.png)
